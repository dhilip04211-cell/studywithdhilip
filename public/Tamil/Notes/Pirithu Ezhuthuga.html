புணர்ச்சி — அடிப்படைக் கருத்துகள்

Punarchchi — Basic Concepts

புணர்ச்சி என்றால் என்ன?
இரண்டு சொற்களுக்கு இடையில் நிகழ்வது புணர்ச்சி. நிலைமொழியோடு அதைத் தொடர்ந்து வரும் வருமொழி இணைவதைப் புணர்ச்சி என்கிறோம். இரு மொழிகளுக்கு இடையே நிகழ்வதால் இது சொற்களின் சந்திப்பு ஆகும்.

🔵 நிலைமொழி (Nilaemozhi)

முதலில் நிற்கும் சொல்
எ.கா: தமிழ் + அன்னை → தமிழ் = நிலைமொழி

🟢 வருமொழி (Varumozhi)

இரண்டாவதாக வரும் சொல்
எ.கா: தமிழ் + அன்னை → அன்னை = வருமொழி

எழுத்துகளின் வகைப்பாடு (புணர்ச்சி அடிப்படையில்)

வகைவிளக்கம்எ.கா
உயிரீறுநிலைமொழியின் இறுதி எழுத்து உயிர் ஆக நிற்கும்மணி (இ — உயிர்)
மெய்யீறுநிலைமொழியின் இறுதி எழுத்து மெய் ஆக நிற்கும்பொன் (ன் — மெய்)
உயிர்முதல்வருமொழியின் முதல் எழுத்து உயிர் ஆக நிற்கும்இலை (இ — உயிர்)
மெய்ம்முதல்வருமொழியின் முதல் எழுத்து மெய் ஆக நிற்கும்நிலம் (ந் — மெய்)

சொற்களின் அடிப்படையில் எழுத்துப் புணர்ச்சி — 4 வகை

எண்ஈற்றெழுத்து + முதலெழுத்துஎ.கா
1உயிர் + உயிர்மலை + அருவி
2மெய் + உயிர் / உயிர் + மெய்தமிழ் + அன்னை / தென்னை + மரம்
3உயிர் + மெய்தேன் + மழை
4மெய் + மெய்—

சொற்களின் இலக்கண அடிப்படை வகைகள்

வ.எண்நிலைமொழி + வருமொழிஎ.காபுணர்ச்சி
1பெயர் + பெயர்கனி + சாறுகனிச்சாறு
2பெயர் + வினைதமிழ் + படிதமிழ்ப்படி
3வினை + வினைநடந்து + செல்நடந்துசெல்
4வினை + பெயர்படித்த + நூல்படித்த நூல்
இடைச்சொல்லும், உரிச்சொல்லும் நிலைமொழியாகவும், வருமொழியாகவும் வந்தாலும் அவற்றின் புணர்ச்சி எழுத்துகளின் புணர்ச்சியே ஆகும்.
🔀

புணர்ச்சியின் வகைகள்

Types of Punarchchi

✅ இயல்புப் புணர்ச்சி

புணர்ச்சியின் போது மாற்றங்கள் எதுவுமின்றி இயல்பாகப் புணர்வது

வாழை + மரம் = வாழை மரம்
செடி + கொடி = செடிகொடி
மண் + மலை = மண்மலை

⚡ விகாரப் புணர்ச்சி

புணர்ச்சியின்போது ஏதேனும் மாற்றம் நிகழ்ந்தால் அது விகாரப் புணர்ச்சி

விகாரப் புணர்ச்சி — 3 வகைகள்

➕
1. தோன்றல்
கல்லூரி + சாலை = கல்லூரிச்சாலை
(புதிய எழுத்து தோன்றியது)
🔄
2. திரிதல்
பல் + பசை = பற்பசை
(எழுத்து வேறொன்றாக மாறியது)
➖
3. கெடுதல்
புறம் + நானூறு = புறநானூறு
(ஓர் எழுத்து மறைந்தது)
தோன்றல் = தோன்றும் | திரிதல் = திரியும் | கெடுதல் = கெடும் — இம்மூன்றும் விகாரம்!
🔗

உடம்படுமெய் புணர்ச்சி (உயிரீற்றுப் புணர்ச்சி)

Udambadu Mei — Vowel-ending junction consonant

உயிரை ஈறாக உடைய சொற்களின்முன் உயிரை முதலாக உடைய சொற்கள் வரும்போது, இரண்டு உயிரொலிகளை ஒன்று சேர்ப்பதற்கு ஒரு மெய் தோன்றும். இதனை உடம்படுமெய் என்பர்.
நன்னூல் விதி — 162
"இ ஈ ஐவழி யவ்வும் ஏனை உயிர்வழி வவ்வும் ஏமுனிவ் விருமையும்
உயிர்வரின் உடம்படு மெய்யென்றாகும்"

யகர உடம்படுமெய் (யவ்வு)

நிலைமொழி ஈறு இ, ஈ, ஐ ஆக இருக்கும்போது → ய் வரும்

நிலைமொழிவருமொழிபுணர்ச்சி
மணி (இ)அழகுமணி + ய் + அழகு = மணியழகு
தீ (ஈ)எரிதீ + ய் + எரி = தீயெரி
ஓடை (ஐ)ஓரம்ஓடை + ய் + ஓரம் = ஓடையோரம்

வகர உடம்படுமெய் (வவ்வு)

நிலைமொழி ஈறு இ, ஈ, ஐ தவிர பிற உயிர்கள் ஆக இருக்கும்போது → வ் வரும்

நிலைமொழிவருமொழிபுணர்ச்சி
பல (அ)உயிர்பல + வ் + உயிர் = பலவுயிர்
பா (ஆ)இனம்பா + வ் + இனம் = பாவினம்
குரு (உ)அருள்குரு + வ் + அருள் = குருவருள்

ஏகார ஈறு — யகரமும் வகரமும்

நிலைமொழி ஈறு ஏகாரம் ஆக இருக்கும்போது → ய் அல்லது வ் வரும்

நிலைமொழிவருமொழிஇரு வடிவங்களும்
சே (ஏ)அடிசே + ய் + அடி = சேயடி
சே + வ் + அடி = சேவடி
தே (ஏ)ஆரம்தே + வ் + ஆரம் = தேவாரம்
இவனே (ஏ)அவன்இவனே + ய் + அவன் = இவனேயவன்
⭐ முக்கிய விதி சுருக்கம்
• நிலைமொழி ஈறு இ, ஈ, ஐ → யகரம் உடம்படுமெய்யாக வரும்
• நிலைமொழி ஈறு ஏனைய உயிர்கள் → வகரம் உடம்படுமெய்யாக வரும்
• நிலைமொழி ஈறு ஏகாரம் → யகரமும் வகரமும் வரும்
• குற்றியலுகரம் ஈறாக இருப்பின் இவ்விதி பொருந்தாது
🔤

குற்றியலுகரப் புணர்ச்சி

Kurriyalugarap Punarchchi

குற்றியலுகர ஈறு — 6 வகைகள்

முக்கிய அறிவிப்பு
சார்பெழுத்துகளுள் குற்றியலுகரத்தின் ஆறு வகைகளும் சொல்லின் இறுதியில் மட்டுமே வரும். ஆய்தம் சொல்லின் முதலிலோ இறுதியிலோ வராது.
வகைநிலைமொழிவருமொழிபுணர்ச்சி
நெடில்தொடர்வீடுஇல்லைவீடில்லை
உயிர்த்தொடர்முரடுகாளைமுரட்டுக்காளை
வன்தொடர்அச்சுபலகைஅச்சுப்பலகை
மென்தொடர்பஞ்சுபொதிபஞ்சுப்பொதி
இடைத்தொடர்மார்புகூடுமார்புக்கூடு
ஆய்தத்தொடர்எஃகுகம்பிஎஃகுக்கம்பி

குற்றியலுகரப் புணர்ச்சி விதிகள்

① உகரம் மறைதல் (உயிர்முன்)

நன்னூல் 164
"உயிர்வரின் உக்குறள் மெய்விட்டு ஓடும்"

குற்றியலுகர நிலைமொழியின் உகரம் மறைந்து, தான் ஏறிய மெய் வருமொழியின் உயிரோடு சேரும்.

எ.கா: வட்டு + ஆடினான் → வட்ட் + ஆடினான் = வட்டாடினான்

② யவ்வரின் இய்யாம் (யகர முதன்மொழியோடு)

குற்றியலுகரம் யகர முதன்மொழியோடு புணரும்போது உகரம் இகரமாகத் திரியும்

எ.கா: கொக்கு + யாது = கொக்கியாது (கு → கி)

③ ஒற்று இரட்டல் (நெடில்தொடர் & உயிர்த்தொடர்)

நன்னூல் 183
"நெடிலோடு உயிர்த்தொடர்க் குற்றுகரங்களுள் புறஒற்று இரட்டும் வேற்றுமை மிகவே"
நிலைமொழிவருமொழிபுணர்ச்சி
வீடுதோட்டம்வீடு → வீட்டு + தோட்டம் = வீட்டுத்தோட்டம்
காடுமரம்காடு → காட்டு + மரம் = காட்டுமரம்
சோறுபானைசோறு → சோற்று + பானை = சோற்றுப்பானை
ஆறுநீர்ஆறு → ஆற்று + நீர் = ஆற்றுநீர்
வயிறுபசிவயிறு → வயிற்று + பசி = வயிற்றுப்பசி

முற்றியலுகரப் புணர்ச்சி

நிலைமொழியின் இறுதியில் உள்ள முற்றியலுகரமும் குற்றியலுகரத்தைப் போலவே தான் ஏறிய மெய்யை நிறுத்தி உகரம் மறையும்.

எ.கா: வரவு + அறிந்தான் → வரவ் + அறிந்தான் → வரவறிந்தான்

"முற்றும் அற்று ஒரோவழி" — நன்னூல் 164
🌟

பண்புப்பெயர்ப் புணர்ச்சி

Pannbu Peyar Punarchchi — 8 Types

பண்புப்பெயர்ச்சொல் நிலைமொழியாக நின்று வருமொழியோடு புணரும்போது ஏற்படும் மாற்றங்கள். மை விகுதி கெட்டு புணர்வது இயல்பு.
நன்னூல் 136
"ஈறு போதல் இடைஉகரம் இய்யாதல்
ஆதி நீடல் அடிஅகரம் ஐ ஆதல்
தன்னொற்று இரட்டல் முன்நின்ற மெய்திரிதல்
இனம் மிகல் இனையவும் பண்பிற்கு இயல்பே"

1. ஈறுபோதல்

மை விகுதி கெட்டுப் புணரும்

பெருமை + வழி → பெருவழி

2. இடையுகரம் இய்யாதல்

மை கெட்டு, உகரம் இகரமாகத் திரியும்

பெருமை + அன் → பெரு → பெரி + ய் + அன் = பெரியன்

3. ஆதிநீடல்

மை கெட்டு, முதல் குறில் நெடிலாகும்

முதுமை + ஊர் → முது → மூது → மூதூர் = மூதூர்

4. அடி அகரம் ஐ ஆதல்

மை கெட்டு, முதல் அகரம் ஐகாரமாகும்

பசுமை + தமிழ் → பசு → பைசு → பை + தமிழ் = பைந்தமிழ்

5. தன்னொற்று இரட்டல்

மை கெட்டு, சொல்லின் ஒற்று இரட்டிக்கும்

நெடுமை + இலை → நெடு → நெட்டு → நெட்டிலை

6. முன்னின்ற மெய் திரிதல்

மை கெட்டு, முன் மெய் வேறாகத் திரியும்

செம்மை + தமிழ் → செம் + தமிழ் = செந்தமிழ்

7. இனம் மிகல்

மை கெட்டு, இன மெல்லின மெய் தோன்றும்

கருமை + கடல் → கரு + கடல் = கருங்கடல்

8. இனையவும்

மேற்சொன்ன விதிகள் பொருந்தாத இடங்களில்

பசுமை + தளிர் → பை + தளிர் = பைந்தளிர்

இயல்பீறு / விதியீறு புணர்ச்சி

"இயல்பினும் விதியினும் நின்ற உயிர்முன் க ச த ப மிகும்" — நன்னூல் 165
நிலைமொழிவருமொழிபுணர்ச்சி

மகர ஈற்றுப் புணர்ச்சி (மவ்வீறு)

நன்னூல் 219
"மவ்வீறு ஒற்றழிந்து உயிரீறு ஒப்பவும் வன்மைக்கு இனமாத் திரிபவும் ஆகும்"
வகைநிலைமொழிவருமொழிபுணர்ச்சி
மகரம் கெடுதல்பாடம்வேளைபாடவேளை

Direction Word & Pala/Sila Punarchchi

திசைப்பெயர் புணர்ச்சி

திசைப் பெயர்களோடு வேறு சொற்கள் புணரும்போது நிலைமொழியின் இறுதியில் உள்ள உயிர்மெய்யெழுத்தும் (கு) ககர ஒற்றும் (க்) நீங்கிப் புணரும். பின் றகரம் னகரமாகவோ (ற் → ன்) லகரமாகவோ (ற் → ல்) திரியும்.
புணர்ச்சிமாற்றம்விளக்கம்
வடக்கு + மேற்கு = வடமேற்குஈற்று (கு) + (க்) நீங்கியதுறகரம் நேரடியாக கெட்டது
தெற்கு + கிழக்கு = தென்கிழக்குற் → ன்றகரம் னகரமாகத் திரிந்தது
மேற்கு + நாடு = மேல்நாடுற் → ல்றகரம் லகரமாகத் திரிந்தது

பல சில புணர்ச்சி — 3 வழிகள்

வழிபல + பலசில + சில
அ) இயல்புபலபலசிலசில
ஆ) வல்லெழுத்து மிகுதல்பலப்பலசிலச்சில
இ) அகரம் கெட்டு றகரம் திரிதல்பற்பலசிற்சில
பிற சொற்களுடன்
பல + கலை = பலகலை (கெடாமல்) அல்லது பல்கலை (கெட்டு)

Consonant Ending Punarchchi & Special Types

ண, ன ஈற்று புணர்ச்சி

ஈறுவருமொழி முதல்மாற்றம்எ.கா
ண்வல்லின மெய்ண் → ட்மண் + கலம் = மட்கலம்
ன்வல்லின மெய்ன் → ற்பொன் + குடம் = பொற்குடம்

ல, ள ஈற்றுப் புணர்ச்சி

ஈறுவல்லின முன்மெல்லின முன்
ல்ல் → ற் (வேல் + காளை = வேற்காளை)ல் → ன் (கயல் + முள் = கயன்முள்)
ள்ள் → ட் (திரைகவுள் + பயனில் = திரைகவுட்பயனில்)ள் → ண் (அருள் + மொழி = அருண்மொழி)

னல முன்னும் ணள முன்னும் — தகர, நகர திரிதல்

ஈறுவருமொழிதகரம் → எ.கா
ன், ல்தத → றபொன் + தீது = பொன்றீது
ன், ல்நந → னபொன் + நன்று = பொன்னன்று
ண், ள்தத → டமண் + தீது = மண்டீது
ண், ள்நந → ணகண் + நீர் = கண்ணீர்

தனிக்குறில்முன் ஒற்று இரட்டல்

நன்னூல் 205
"தனிக்குறில்முன் ஒற்று உயிர்வரின் இரட்டும்"

எ.கா: கல் + அதர் → கல்ல் + அதர் = கல்லதர்

பூப்பெயர்ப் புணர்ச்சி

'பூ' என்னும் சொல் நிலைமொழியாய் நிற்க, வருமொழியின் முதலில் வல்லின மெய்யெழுத்துகள் வரும்போது வல்லினம் மட்டுமன்றி இன மெல்லினமும் மிகும்.
நிலைமொழிவருமொழிஇரு வடிவங்களும்
பூகொடிபூக்கொடி / பூங்கொடி

மெய்ம்மயக்கம் (Mei Mayakkam)

புணர்ச்சியில் வருமொழியில் க, ச, த, ப வந்தால் சில இடங்களில் மீண்டும் அதே எழுத்துத் தோன்றும். இதை "வலி மிகுதல்" என்பர். மெல்லினமும் மிகும்.
புணர்ச்சிமெய்ம்மயக்கம்
மெய் + மயக்கம்மெய்ம்மயக்கம்
மெய் + ஞானம்மெய்ஞ்ஞானம்
செய் + நன்றிசெய்ந்நன்றி
வேய் + குழல்வேய்ங்குழல்
கூர் + சிறைகூர்ஞ்சிறை
மா + பழம்மாம்பழம்
விள + காய்விளங்காய்
புளி + கறிபுளிங்கறி
