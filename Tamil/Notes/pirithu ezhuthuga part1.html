1

நிலைமொழி – வருமொழி

அடிப்படைக் கருத்துகள்

சொற்களைப் பிரித்து எழுதுவதற்கும், சேர்த்து எழுதுவதற்கும் இரண்டு சொற்களில் ஏற்படும் மாற்றங்களை விளக்கும் புணர்ச்சி இலக்கணம் குறித்துத் தெரிந்து கொள்ள வேண்டும்.

★

இரண்டு சொற்களுக்கு இடையில் நிகழ்வது – புணர்ச்சி

★

இரு மொழிகளுக்கு முதலில் நிற்கும் நிலைமொழியோடு இடையே நிகழ்வது தான் புணர்ச்சி. அதைத் தொடர்ந்து வரும் வருமொழி இணைவதைப் புணர்ச்சி என்கிறோம்.

★

புணர்ச்சியில் நிலைமொழியின் முதலெழுத்தைப் பொறுத்து இறுதி எழுத்தைப் பொறுத்து உயிரீறு, மெய்யீறு எனவும் வருமொழியின் உயிர்முதல், மெய்ம்முதல் எனவும் பிரிக்கலாம்.

★

நிலைமொழியின் இறுதி எழுத்தும் வருமொழியின் முதலெழுத்தும் புணர்ச்சிக்குரியன.

எ.கா (Examples)
தமிழ்+அன்னை=தமிழன்னை (நிலைமொழி + வருமொழி)
தமிழ்+அரசி=தமிழரசி
நாடு+பண்=நாட்டுப்பண்
உயிரீறு
★

நிலைமொழியின் இறுதி எழுத்து உயிர்மெய்யாக இருந்தாலும் அதன் இறுதியில் நிற்கும் வடிவம் உயிர் என்பதால் அது உயிரீறு எனப்படும்.

எ.கா
மணி (ண் + இ) + மாலை = மணிமாலை (இ – உயிரீறு)
மெய்யீறு
★

நிலைமொழியின் இறுதி எழுத்து மெய்யாக இருந்தால் அது மெய்யீறு எனப்படும்.

எ.கா
பொன்+வண்டு=பொன்வண்டு(ன் – மெய்யீறு)
உயிர்முதல்
★

வருமொழியின் முதலெழுத்து உயிரெழுத்தாக இருந்தால் அது உயிர்முதல் எனப்படும்.

எ.கா
வாழை+இலை=வாழையிலை(இ – உயிர்முதல்)
மெய்ம்முதல்
★

வருமொழியின் முதலெழுத்து உயிர்மெய்யாக இருந்தாலும் அதன் முதலில் நிற்கும் வடிவம் மெய் என்பதால் அது மெய்ம்முதல் எனப்படும்.

எ.கா
தமிழ்+நிலம்=தமிழ்நிலம்
தமிழ்+(ந் + இ) லம்=தமிழ்நிலம்(ந் – மெய்ம்முதல்)
★

புணர்ச்சி என்பது எழுத்துகளின் சந்திப்பாகவும், சொற்களின் சந்திப்பாகவும் அமைகிறது.

★

புணர்ச்சி என்பது எழுத்துகளும், சொற்களும் ஒலிக்கூறுகளாகவும் பொருள்கூறுகளாகவும் சந்திக்கும் நிகழ்வு.

2

எழுத்துகளின் அடிப்படையில் புணர்ச்சி

4 வகை
★

சொற்புணர்ச்சியில் நிலைமொழி இறுதி எழுத்தாகவும், வருமொழி முதலெழுத்தாகவும் எழுத்துகள் சந்திக்கும் முறையை நான்கு விதமாகப் பிரிக்கலாம்.

வ.எண் ஈற்றெழுத்து + முதலெழுத்து எ.கா. சந்திப்பு
1.உயிர் + உயிர்மலை + அருவிஐ + அ
2.மெய் + உயிர்தமிழ் + அன்னைழ் + அ
3.உயிர் + மெய்தென்னை + மரம்ஐ + ம்
4.மெய் + மெய்தேன் + மழைன் + ம்
★

மேலும் இப்புணர்ச்சியை நிலைமொழி இறுதி எழுத்து, வருமொழி முதல் எழுத்து என்ற அடிப்படையில் நான்காகப் பிரிக்கலாம்.

வகைஎ.கா.புணர்ந்த சொல்
உயிர்முன் உயிர் மணி (ண்+இ) + அடி மணியடி
உயிர்முன் மெய் பனி (ன்+இ) + (க்+ஆ) காற்று பனிக்காற்று
மெய்ம்முன் உயிர் ஆல் + இலை ஆலிலை
மெய்ம்முன் மெய் மரம் + (க்+இ) கிளை மரக்கிளை
புணர்மொழியின் இயல்பு – எழுத்து வகையால் சொற்கள் 4 வகைப்படும்
வகைஎ.கா.
உயிரீறுகலை (ல் + ஐ) + அழகு
மெய்யீறுமண் + குடம்
உயிர் முதல்வாழை + இலை
மெய்ம்முதல்வாழை + (ம் + அ) மரம்
3

சொற்களின் அடிப்படையில் புணர்ச்சி

4 வகை
★

இலக்கண வகையால் சொற்கள் நான்கு வகைப்படும் அவை:
1. பெயர்ச்சொல்    2. வினைச்சொல்    3. இடைச்சொல்    4. உரிச்சொல்

வ.எண்நிலைமொழி + வருமொழிஎ.கா.புணர்ச்சி
1.பெயர் + பெயர்கனி + சாறுகனிச்சாறு
2.பெயர் + வினைதமிழ் + படிதமிழ்ப்படி
3.வினை + வினைநடந்து + செல்நடந்துசெல்
4.வினை + பெயர்படித்த + நூல்படித்த நூல்
★

இவற்றில் இடைச்சொல்லும், உரிச்சொல்லும் பெயரையும், வினையையும் சார்ந்தே வருகின்றன.

★

இடைச்சொல்லும், உரிச்சொல்லும் நிலைமொழியாகவும், வருமொழியாகவும் வந்தாலும் அவற்றின் புணர்ச்சி, எழுத்துகளின் புணர்ச்சியே ஆகும்.

4

இயல்புப் புணர்ச்சி & விகாரப் புணர்ச்சி

2 வகை புணர்ச்சி
★

புணர்ச்சியில் நிலைமொழியும் வருமொழியும் அடையும் மாற்றங்களின் அடிப்படையில் புணர்ச்சியை இருவகைப் படுத்தலாம்:
1. இயல்புப் புணர்ச்சி 2. விகாரப் புணர்ச்சி

🟢 இயல்புப் புணர்ச்சி

புணர்ச்சியின் போது மாற்றங்கள் எதுவுமின்றி இயல்பாகப் புணர்வது இயல்பு புணர்ச்சி எனப்படும்.

வாழை + மரம் = வாழை மரம்
செடி + கொடி = செடிகொடி
மண் + மலை = மண்மலை

🔴 விகாரப் புணர்ச்சி

புணர்ச்சியின்போது ஏதேனும் மாற்றம் நிகழ்ந்தால் அது விகாரப் புணர்ச்சி எனப்படும்.

மூன்று வகைகள்: தோன்றல் | திரிதல் | கெடுதல்

கல்லூரி + சாலை = கல்லூரிச்சாலை (தோன்றல்)
பல் + பசை = பற்பசை (திரிதல்)
புறம் + நானூறு = புறநானூறு (கெடுதல்)
5

குற்றியலுகர ஈறு

6 வகைகள்
★

சார்பெழுத்துகளுள் ஆய்தம் சொல்லின் முதலிலோ இறுதியிலோ வராது.

★

குற்றியலுகரமும் (நுந்தை தவிர) குற்றியலிகரமும் இக்காலத்தில் சொல்லின் முதலில் வாரா.

★

சார்பெழுத்துகளுள் குற்றியலுகரத்தின் ஆறு வகைகளும் சொல்லின் இறுதியில் வரும்.

★

குற்றியலுகர ஈற்றுடன் வரும் நிலைமொழி குற்றியலுகர ஈறு (அ) குற்றியலுகர நிலைமொழி எனப்படும்.

எ.கா
வீடு+இல்லை=வீடில்லை
நிலைமொழி + வருமொழிபுணர்ந்த சொல்வகை
வீடு + இல்லைவீடில்லைநெடில்தொடர்க் குற்றியலுகரம்
முரடு + காளைமுரட்டுக்காளைஉயிர்த்தொடர்க் குற்றியலுகரம்
அச்சு + பலகைஅச்சுப்பலகைவன்தொடர்க் குற்றியலுகரம்
பஞ்சு + பொதிபஞ்சுப்பொதிமென்தொடர்க் குற்றியலுகரம்
மார்பு + கூடுமார்புக்கூடுஇடைத்தொடர்க் குற்றியலுகரம்
எஃகு + கம்பிஎஃகுக்கம்பிஆய்தத் தொடர்க் குற்றியலுகரம்
6

புணர்ச்சி விதிகள்

பயன்கள்
★

சொற்புணர்ச்சியின்போது நிலைமொழி இறுதியிலும் வருமொழி முதலிலும் ஏற்படும் மாற்றங்களைச் சுருங்கச் சொல்லும் வரையறைகளைப் புணர்ச்சி விதிகள் என்பர்.

புணர்ச்சி விதிகளின் பயன்கள்:
  • மொழியைப் பிழையின்றிக் கையாள.
  • பாடலடிகளைப் பொருள் உணர்வுக்கு ஏற்ற வகையில் பிரித்தறிய.
  • மொழி ஆளுமையைப் புரிந்து கொள்ள.
7

அ) உடம்படுமெய் புணர்ச்சி

உயிரீற்றுப் புணர்ச்சி

"இ ஈ ஐவழி யவ்வும் ஏனை
உயிர்வழி வவ்வும் ஏமுனிவ் விருமையும்
உயிர்வரின் உடம்படு மெய்யென்றாகும்"

– நன்னூல் 162
★

உயிரை ஈறாக உடைய சொற்களின்முன் உயிரை முதலாக உடைய சொற்கள் வந்து சேரும்; அப்போது சொற்கள் சேராமல் தனித்து நிற்கும்; ஒன்று சேராத உயிரொலிகளை ஒன்று சேர்ப்பதற்கு அங்கு ஒரு மெய் தோன்றும். இதனை உடம்படுமெய் என்பர்.

நிலைமொழி ஈறு இ, ஈ, ஐ → அவற்றின்முன் உயிர்முதல் சொல் சேரும்போது யகரம் உடம்படுமெய்யாக வரும்.

யகர உடம்படுமெய் – எ.கா
மணி+அழகு= மணி + ய் + அழகு=மணியழகு
தீ+எரி= தீ + ய் + எரி=தீயெரி
ஓடை+ஓரம்= ஓடை + ய் + ஓரம்=ஓடையோரம்

இ, ஈ, ஐ தவிர, பிற உயிரெழுத்துகள் நிலைமொழி ஈறாக வரும்போது → வகர மெய் தோன்றும்.

வகர உடம்படுமெய் – எ.கா
பல+உயிர்= பல + வ் + உயிர்=பலவுயிர்
பா+இனம்= பா + வ் + இனம்=பாவினம்

நிலைமொழி ஈறு ஏகாரம் ஆயின் → யகரமோ வகரமோ (இரு உடம்படுமெய்களும்) வரும்.

ஏகார ஈறு – எ.கா
சே+அடி= சேயடி அல்லது சேவடி
தே+ஆரம்= தே + வ் + ஆரம்=தேவாரம்
இவனே+அவன்= இவனே + ய் + அவன்=இவனேயவன்
நிலைமொழி ஈறுஉடம்படுமெய்
இ, ஈ, ஐயகரம்
ஏனைய உயிர்கள்வகரம்
ஏகாரம்யகரம் + வகரம் (இரண்டும்)
★

நிலைமொழி இறுதியில் ஈறாக இருந்தும் குற்றியலுகரமாக இருப்பின் இவ்விதி பொருந்தாது.

ஆ) குற்றியலுகரப் புணர்ச்சி

குற்றியலுகரச்சொற்களில் நிலைமொழியின் இறுதியில் உள்ள குற்றியலுகரம் வருமொழியின் முதலில் உள்ள உயிரெழுத்துடன் புணரும்போது, தான் ஏறிய மெய்யை நிறுத்தி உகரம் மறையும்.

2

★

நிலைமொழியிறுதிக் குற்றியலுகரம், யகர முதன்மொழியோடு புணரும் பொழுது உகரம் இகரமாகத் திரியும்.

கொக்கு+யாது= கொக்கியாது(கு → கி)

விதி – யவ்வரின் இய்யாம்

★

சில இடங்களில் நிலைமொழி இறுதி முற்றியலுகரம் உயிர் முதன்மொழியோடு புணரும்போது தான் ஊர்ந்து வந்த மெய்யை விட்டு மறையும். பின் உடல்மேல் உயிர்வந்து ஒன்றுவது இயல்பே என்றும் விதிப்படி புணரும்.

எ.கா
கதவு+அடைப்பான்= கதவ் + அடைப்பான்= கதவடைப்பான்

★

"நெடிலோடு உயிர்த்தொடர்க் குற்றுகரங்களுள் புறஒற்று இரட்டும் வேற்றுமை மிகவே"

ஆறுநீர்ஆற்று + நீர்ஆற்றுநீர்
வயிறுபசிவயிற்று + பசிவயிற்றுப்பசி

புணர்ச்சி சுருக்க அட்டவணை

இயல்பு புணர்ச்சி

விகாரம்

கல் + சிலை = கற்சிலை

உடம்படுமெய்

அவை நீங்கிய பின் நிலைமொழியின் இறுதியில் உள்ள றகரம் இனகரமாகவோ (ன்) லகரமாகவோ (ல்) திரிந்தும் புணரும்.

வடக்கு+மேற்கு= வடமேற்கு (ஈற்று உயிர்மெய்யும் (கு), ககர ஒற்றும் (க்) நீங்கிப் புணர்ந்தது)

மேற்கு+நாடு= மேல்நாடு (ஈற்று உயிர்மெய் (கு) கெட்டு றகரம் லகரமாகத் (ற் → ல்) திரிந்து புணர்ந்தது)

"திசையோடு திசையும் பிறவும் சேரின்
நிலையீற்றுயிர்மெய் கல்வொற்று நீங்கனும்
றகரம் னலவாத் திரிதலும் ஆம்பிற"

பல சில – புணர்ச்சி

★

இ) நிலைமொழி இறுதியிலுள்ள அகரம் கெட எஞ்சிநிற்கும் லகரம் றகரமாகத் திரிந்து புணரும்.

ணகர, னகர ஈறு

★

வேற்றுமைப் புணர்ச்சியில் நிலைமொழியீற்று ணகரமும் னகரமும் முறையே வல்லின முதன்மொழியோடு புணரும்போது டகரமாகவும் றகரமாகவும் திரியும்.

ஐ) பண்புப்பெயர்ப் புணர்ச்சி

8 விதிகள்
★

பண்புப்பெயர்ச்சொல் நிலைமொழிச் சொல்லாக நின்று வருமொழிச்சொல்லுடன் புணரும்போது பின்வரும் மாற்றங்களை அடையும்.

1. ஈறுபோதல்
★

நிலைமொழியின் இறுதியில் உள்ள 'மை' கெட்டுப் புணரும்.

எ.கா
1

பெருமை + வழி

2

ஈறுபோதல் விதிப்படி மை விகுதி கெட்டு → பெருவழி

எ.கா – பெரியன்
1

பெரியன் = பெருமை + அன்

2

பெரு + அன் → ஈறுபோதல் (மை)

3

பெரி + அன் → இடை உகரம் இய்யாதல் (ரு → ரி)

4

பெரி + ய் + அன் → உடம்படுமெய் 'ய்' இடையில் தோன்றும் → பெரியன்

3. ஆதிநீடல்
★

மை விகுதி கெட்டு நிலைமொழிச் சொல்லின் முதலில் உள்ள குறில் எழுத்து நெடில் எழுத்தாய் மாறும்.

எ.கா – மூதூர்
1

மூதூர் = முதுமை + ஊர்

2

முது + ஊர் → ஈறுபோதல் (மை)

3

மூது + ஊர் → ஆதிநீடல் (மு → மூ)

4

மூத் + ஊர் → உயிர்வரின் உக்குறள் மெய்விட்டு ஓடும்

5

மூதூர் – உடல்மேல் உயிர்வந்து ஒன்றுவது இயல்பே (த் + ஊ = தூ)

4. அடி அகரம் ஐ ஆதல்
★

மை விகுதி கெட்டு நிலைமொழிச் சொல்லின் முதலில் உள்ள அகரம் ஐகாரமாக மாறும்.

எ.கா – பைந்தமிழ்
1

பைந்தமிழ் = பசுமை + தமிழ்

2

பசு + தமிழ் → ஈறுபோதல் (மை)

3

பைசு + தமிழ் → அடி அகரம் ஐ ஆதல் (ப → பை)

4

பை + தமிழ் → இனையவும் ('சு' கெட்டது)

5

"இனமிகல்" என்னும் விதிப்படி பைந்தமிழ் எனப் புணர்ந்தது

5. தன்னொற்று இரட்டல்
★

மை விகுதி கெட்டு நிலைமொழிச் சொல் ஒற்று இரட்டிக்கும்.

5

நெட்டிலை – உடல்மேல் உயிர்வந்து ஒன்றுவது இயல்பே (ட் + இ = டி)

வெற்றிலை: வெறுமை + இலை → வெற்று + இலை → வெற்றிலை
6. முன்னின்ற மெய் திரிதல்
★

ஈறுபோதல் விதியின்படி மை கெட்டு நிலைமொழியின் இறுதி எழுத்து மகர மெய்யாக இருந்தால் முன் நின்ற மெய் திரிதல் விதியின்படி புணரும்.

2

3

5

"இனமிகல்" என்னும் விதிப்படி பைந்தளிர் எனப் புணர்ந்தது

மெய்ம்மயக்கம்

★

புணர்ச்சியில் இரு சொற்கள் இணையும்போது வருமொழியில் க, ச, த, ப வந்தால் சில இடங்களில் மீண்டும் அதே எழுத்துத் தோன்றும். இதை "வலி மிகுதல்" என்பர்.

★

இது போன்றே சில இடங்களில் மெல்லினமும் மிகுதல் உண்டு. குறிப்பாக, ங, ஞ, ந, ம என்ற நான்கு எழுத்துகளும் இவ்வாறு மிகும்.

1. 'ய' கர ஈற்றுச் சொற்கள் முன் மெல்லினம் மிகும்
புணர்ச்சிபுணர்ந்த சொல்
மெய் + மயக்கம்மெய்ம்மயக்கம்
மெய் + ஞானம்மெய்ஞ்ஞானம்
செய் + நன்றிசெய்ந்நன்றி
வேற்றுநிலை மெய்ம்மயக்கத்தில் ய, ர, ழ முன்னர் மெல்லினம் மிகும்
புணர்ச்சிபுணர்ந்த சொல்
வேய் + குழல்வேய்ங்குழல்
கூர் + சிறைகூர்ஞ்சிறை
பாழ் + கிணறுபாழ்ங்கிணறு
2. 'புளி' என்னும் சுவைப் பெயர்முன்னர் வல்லெழுத்து மட்டுமன்றி மெல்லினமும் மிகும்
புணர்ச்சிபுணர்ந்த சொல்
புளி + கறிபுளிங்கறி
புளி + சோறுபுளிஞ்சோறு
3. உயிரெழுத்தை இறுதியில் கொண்ட மரப் பெயர்களுக்கு முன்னர் மெல்லினம் மிகும்
புணர்ச்சிபுணர்ந்த சொல்
மா + பழம்மாம்பழம்
விள + காய்விளங்காய்
4. 'பூ' என்னும் பெயர் முன்னர் வல்லினத்தோடு மெல்லினமும் மிகும்
புணர்ச்சிபுணர்ந்த சொல்
பூ + கொடிபூங்கொடி
பூ + சோலைபூஞ்சோலை
பூ + தொட்டிபூந்தொட்டி
