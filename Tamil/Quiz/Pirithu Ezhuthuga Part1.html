கவனிக்கவும்: ஒவ்வொரு வினாவிலும் சரியான விடையைத் தேர்ந்தெடுக்கவும். விடை உடனடியாக காட்டப்படும். மேலே உள்ள மதிப்பெண் பலகையில் உங்கள் முன்னேற்றம் தெரியும். அனைத்து வினாக்களும் TAF IAS Academy புத்தகத்தில் இருந்து எடுக்கப்பட்டவை.
1
அடிப்படை பிரித்து எழுதுதல் — பக்கம் 13–14
வினாக்கள் 1–30
வினா 1
"முடிவிலா" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: முடிவு + இலா — 'முடிவு' என்ற பெயர்ச்சொல்லுடன் 'இலா' என்ற இடைச்சொல் சேர்வதால் உகரம் கெட்டு முடிவிலா ஆகும்.
வினா 2
"இரவெல்லாம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இரவு + எல்லாம் — 'இரவு' என்ற சொல்லில் உகரம் கெட்டு 'எல்லாம்' சேரும்போது இரவெல்லாம் ஆகும்.
வினா 3
"காப்பாற்றென்று" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: காப்பாற்று + என்று
வினா 4
"வேங்கடமாமலை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வேங்கடம் + மா + மலை — மூன்று சொற்கள் சேர்ந்து வரும் தொடர்மொழி.
வினா 5
"வீட்டிருந்து" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வீடு + இருந்து
வினா 6
"இணரெரி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இணர் + எரி
வினா 7
"உயர்வுள்ளல்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: உயர்வு + உள்ளல்
வினா 8
"நல்வினை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நன்மை + வினை — 'நன்மை' என்பது திரிந்து 'நல்' ஆகி வல்லொற்று மிகும்போது 'நல்வினை' ஆகும்.
வினா 9
"பேயென்றாள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பேய் + என்றாள்
வினா 10
"ஒட்டலரிது" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஒட்டல் + அரிது
வினா 11
"நீயென்றேன்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நீ + என்றேன்
வினா 12
"ஊக்கமுடையான்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஊக்கம் + உடையான்
வினா 13
"நல்குரவொழிய" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நல்குரவு + ஒழிய
வினா 14
"அரும்பொருள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: அருமை + பொருள் — 'அருமை' திரிந்து 'அரும்' ஆகும்.
வினா 15
"காராண்மை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கார் + ஆண்மை
வினா 16
"திருவருள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: திரு + அருள்
வினா 17
"வெள்ளத்தனைய" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வெள்ளம் + அத்து + அனைய — மூன்று சொற்கள் சேர்ந்த தொடர்மொழி.
வினா 18
"நரியென்றாள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நரி என்றாள்
வினா 19
"உலகத்துயர்ந்த" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: உலகத்து + உயர்ந்த
வினா 20
"பொற்கோட்டுமேரு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பொன் + கோடு + மேரு
வினா 21
"நல்லறம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நன்மை + அறம்
வினா 22
"இவ்விரண்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இவ் + இரண்டு
வினா 23
"புத்தேளுலகு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: புத்தேன் + உலகு
வினா 24
"பல்பொருணீங்கிய" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பல + பொருள் + நீங்கிய
வினா 25
"தாழ்வின்றி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தாழ்வு + இன்றி
வினா 26
"அங்கண்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: அம் + கண்
வினா 27
"வண்பயன்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வண்மை + பயன் — வண்மை திரிந்து வண் ஆகும்.
வினா 28
"ஈராறாண்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இரண்டு + ஆறு + ஆண்டு — பன்னிரண்டு ஆண்டு என்பதற்கான தொகைமொழி.
வினா 29
"பெருங்குணம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பெருமை + குணம்
வினா 30
"வீட்டிருந்து" என்பது போல் "உடனெய்தும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: உடன் + எய்தும்
2
நடுத்தர வினாக்கள் — பக்கம் 14–16
வினாக்கள் 31–55
வினா 31
"கேழிலாள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கேழ் + இலாள்
வினா 32
"வெங்கதிர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வெம்மை + கதிர் — வெம்மை திரிந்து வெம் ஆகி, வல்லொற்று மிக வெங் ஆகும்.
வினா 33
"மாசுண்ட" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மாசு + உண்ட
வினா 34
"செவ்வேள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: செம்மை + வேள்
வினா 35
"கருநிறத்தரக்கியர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கருமை + நிறத்து + அரக்கியர்
வினா 36
"அடுப்பகத்தேற்றி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: அடுப்பு + அகத்து + ஏற்றி
வினா 37
"விடியளவும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: விடி + அளவும்
வினா 38
"தடிந்தழிதல்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தடிந்து + இழிதல்
வினா 39
"சேவடி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: செம்மை + அடி — செம்மை திரிந்து சே ஆகி சேவடி ஆகும்.
வினா 40
"வெவ்விருப்பாணி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வெம்மை + இரும்பு + ஆணி
வினா 41
"வடமலை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வடக்கு + மலை
வினா 42
"தென்மலை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தெற்கு + மலை
வினா 43
"கனகமகாமேரு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கனகம் + மகா + மேரு
வினா 44
"நாஞ்சினாட்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நாஞ்சில் + நாட்டு
வினா 45
"மலலையிங்கள்" — இதைப் போல் "மலையிங்கள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மலை + இங்கள் (மலையிங்கள் = மலை + இங்கள்)
வினா 46
"பாழ்மலை" என்பதைப் போல் "மலலையிங்கள்" என்ற மூலச்சொல்லில் "மலையிங்கள்" = மலை + இங்கள். அதுபோல் "பழம்பாடல்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பழமை + பாடல்
வினா 47
"திந்தமிழ்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தீம் + தமிழ்
வினா 48
"தமிழ்முகுதவு" என்பதைப் போல் "தமிழ்முகுதவு" = தமிழ் + உதவு என்றாகும். அதேபோல் "பழம்பெரும்பூமி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பழமை + பெருமை + பூமி
வினா 49
"திருவுள்ளத்திலழிகு" என்பதைப் போன்று "திருவுள்ளத்திலழிகு" ← இதனில் "கோயிற்கேற்றும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கோயிற்கு + ஏற்றும் (கோயிற்கேற்றும் = கோயிற்கு + ஏற்றும்)
வினா 50
"குழற்காடேந்துமிள" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: குழல் + காடு + ஏந்தும் + இள — குழற்காடேந்துமிள = குழல் + காடு + ஏந்தும் + இள
வினா 51
"செல்லினங்கள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: செல் + இனங்கள்
வினா 52
"வெற்றிக்கொடி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வெற்றி + கொடி
வினா 53
"வெற்றிடமின்றி" என்பதைப் போல் "வெற்பிடமின்றி" — "தோய்வன்ன" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தோய்வு + அன்ன
வினா 54
"தாய்மைஆக" என்பதைப் போல் "தூய்மையாக" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தூய்மை + ஆக
வினா 55
"முடிவெய்துகாறும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: முடிவு + எய்து + காறும்
3
மேம்பட்ட வினாக்கள் — பக்கம் 17–20
வினாக்கள் 56–80
வினா 56
"மலடி" என்பதைப் போல் "மலரடி" என்பதைப் போல் — "எண்ணெய்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: எள் + நெய்
வினா 57
"தண்ணீர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தண்மை + நீர்
வினா 58
"நூற்றாண்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நூறு + ஆண்டு
வினா 59
"அலகிலா" என்பதைப் போல் — "திருக்கசேற்று" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: திரு + சேற்று — திருக்கசேற்று = திரு + சேற்று (திரு+செற்று)
வினா 60
"பூந்தண்கா" என்பதைப் போல் — "பைந்தமிழ்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பசுமை + தமிழ் — பசுமை திரிந்து பைந் ஆகும்.
வினா 61
"வாழ்த்தாதென்னே" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வாழ்த்தா (த) து + என்னே
வினா 62
"நடுவிகந்தாம்" என்ற சொல்லில் இதைப் போல் — "நடுவிகந்தாம்" = நடுவு + இகந்தாம். அதேபோல் "மணிமேகலை" வரிகளில் — "காட்டிலழும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: காட்டில + அழும்
வினா 63
"பாண்டியராயிற்று" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பாண்டியர் + ஆயிற்று
வினா 64
"முச்சந்தி" என்பதைப் போல் — "முத்தமிழ்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மூன்று + தமிழ் — இயல், இசை, நாடகம் என்ற மூன்றும்.
வினா 65
"அரண்ஓரு" என்பதைப் போல் — "அரணாறு" = "அரண் + ஒரு" என்றாகும். அதேபோல் "அரணாறு" என்பதைப் போல் — "செந்தமிழ்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: செம்மை + தமிழ்
வினா 66
"இலக்கியப்பேட்டகம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இலக்கியம் + பேட்டகம்
வினா 67
"வேல்" என்பதைப் போல் — "வேரல்" என்பதைப் போல் — "வேரல்" = வேல் + தல் → "வேரல்" ← இதைப் போல் "வேரல்" → "காடேல்லாம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: காடு + எல்லாம்
வினா 68
"முத்தொள்ளாயிரம்" போன்று — "முத்தொள்ளாயிரம்" = மூன்று + தொள்ளாயிரம். அதுபோல் "முப்பழம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மூன்று + பழம்
வினா 69
"பல்கலைக்கழகம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பல் + கலை + கழகம்
வினா 70
"வென்றிலங்கைவேந்தன்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வென்று + இலங்கை + வேந்தன்
வினா 71
"சிந்தென்று" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: சிந்து + என்று
வினா 72
"சேரவர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: சேர் + ஆவர் (சேரவர் = சேர்+ஆவர்)
வினா 73
"ஒருவர்க்கீது" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஒருவர்க்கு + ஈது
வினா 74
"வேர்க்கோட்பலவின்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வேர் + கோள் + பலவின்
வினா 75
"நிலைமேலேற்ற" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நிலை + மேல் + ஏற்ற
வினா 76
"நாலடியார்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நான்கு + அடி + ஆர் (நாலடியார் = நான்கு + அடி + ஆர்)
வினா 77
"உய்வின்றி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: உய்வு + இன்றி
வினா 78
"கைதொழுதேத்தி" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கை + தொழுது + ஏத்தி
வினா 79
"முக்குணம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மூன்று + குணம்
வினா 80
"நீரகம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நீர் + அகம்

✅ சரியான விடை: பசுமை + இலை
வினா 82
"கலம்பகம்" என்ற சொல்லை பிரிக்கவும்.

வினா 83
"மூதூர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: முதுமை + ஊர் — மூதூர் = முதுமை + ஊர்.
வினா 84
"குருகுமுண்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: குருகும் + உண்டு
வினா 85
"இலக்கியம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இலக்கு + இயம் (வேம்மை = வேண்மை)
வினா 86
"வேரூன்றிய" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வேர் + ஊன்றிய
வினா 87
"நெடுநாவாய்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நெடுமை + நாவாய்
வினா 88
"கரியன்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கருமை + அன்
வினா 89
"தாதுஓவி" என்பதுபோல் "தாதுஓவி" = தாது + ஓவி → "தாதுஓவி" ← இதைப்போல் — "தாதுஓவி" → "பூஞ்சோலை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பூ + சோலை
வினா 90
"நாட்காரணம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நான்கு + காரணம்
வினா 91
"தினேச்சை" என்பதைப் போல் — "தினேச்சை" = தினம் + இச்சை → "தினேச்சை" ← இதைப்போல் — "ஊராண்மை" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஊர் + ஆண்மை
வினா 92
"தவமிரண்டு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: தவம் + இரண்டு
வினா 93
"அரிவையம்பாகத்தான்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: அரிவை + அம்பு + ஆகத்தான்
வினா 94
"நீயால்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நீ + ஆல்
வினா 95
"அண்ணாமலைக்கனுகூலன்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: அண்ணாமலைக்கு + அனுகூலன்
வினா 96
"ஞானத்தத்துவங்கள்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஞானம் + தத்துவங்கள்
வினா 97
"பாம்பெள்ளெனவே" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பாம்பு + எள் + எனவே
வினா 98
"யாரஃதறிந்திசினோர்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: யார் + அஃது + அறிந்திசினோர்
வினா 99
"பயன்விளைப்பாளந்த" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பயன் + விளைப்பாள் + அந்த (பயன்விளைப்பாளந்த)
வினா 100
"மகற்காற்றும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மகன் + கு + ஆற்றும் (மகற்கு = மகன் + கு)
வினா 101
"ஆடும்போதே யிரையும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஆடும்போதே + இரையும்
வினா 102
"பிண்ணாக்கும்உண்டாம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: பிண்ணாக்கும் + உண்டாம் — (பிள் + நாக்கும் + உண்டாம்)
வினா 103
"கணிப்பரிய" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கணிப்பு + அரிய — முடிவெடுக்க இயலாதது என்று பொருள்.
வினா 104
"ஏற்றைகண்டா" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: ஏற்று + கண்டா (இரட்டு + கண்டாண் = இரண்டு + கண்டு + ஆண்)
வினா 105
"மன்னவரன்னவரெல்லாம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: மன்னவர் + அன்னவர் + எல்லாம்
வினா 106
"நமதென்பதறிந்தோம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: நமது + என்பது + அறிந்தோம்
வினா 107
"கன்னலெனத்தகும்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கன்னல் + எனத் + தகும்
வினா 108
"வைய்த்தோ — வைத்தோம் — வைத்தோ?" என்ற கேள்விக்கு — "கூடுதல்விளைவு" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: கூடுதல் + விளைவு
வினா 109
"வியன்உலகம்" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: வியன் + உலகம்
வினா 110
"இரப்பார்க்கீந்து" என்ற சொல்லை பிரிக்கவும்.
✅ சரியான விடை: இரப்பார்க்கு + ஈந்து

வினாக்கள் 111–135
வினா 111
"அமுதென்று" என்ற சொல்லை பிரிக்கவும்.
